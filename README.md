# TEANA – Tea Store Landing Page

A modern landing page for a premium tea store featuring Chinese and Japanese teas.

---

## About the Project

**Teana** is a front-end project created to practice modern web development techniques, including:

- HTML5 semantic markup  
- CSS3 styling and animations  
- Flexbox layouts  
- JavaScript interactions  
- Product card sliders  
- Modern UI design  
- Landing page development  

The website showcases a curated collection of premium Chinese and Japanese teas with product cards, promotional sections, and a clean user interface.

---

## Features

###  Hero Section
- Full-screen banner  
- Background image with overlay  
- Call-to-action button  
- Navigation menu  

###  Tea Catalog
- Chinese tea collection  
- Japanese tea collection  
- Product cards with images, descriptions, and prices  
- Horizontal slider navigation  

###  Promotional Section
- Special discount offer  
- Call-to-action button  
- Custom background image  

###  Footer
- Company information  
- Navigation links  
- Social media links  

###  Animations
- Hover effects on buttons  
- Hover effects on tea cards  
- Smooth transitions
